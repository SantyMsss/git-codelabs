¿Qué es GitHub Education?

GitHub Education es un programa de GitHub diseñado para apoyar a estudiantes, profesores e instituciones educativas con herramientas y recursos gratuitos para el aprendizaje y la enseñanza de desarrollo de software.

Requisitos de elegibilidad

Para poder acceder a GitHub Education y el Student Developer Pack, debes cumplir con ciertos requisitos de elegibilidad.

¿Quién puede aplicar?

  • Estudiantes actualmente inscritos en una institución educativa acreditada
  • Estudiantes de 13 años o mayores
  • Estudiantes de tiempo completo o medio tiempo
  • Estudiantes de secundaria, pregrado, posgrado o doctorado
  • Estudiantes de bootcamps o programas de coding reconocidos

Preparar la documentación

Para una solicitud exitosa, necesitas preparar documentación que pruebe tu estatus estudiantil.

Documentos aceptables

GitHub acepta los siguientes documentos como prueba:

  • Identificación estudiantil actual con fecha de emisión
  • Carta de aceptación o inscripción de la institución
  • Horario de clases actual
  • Transcripción o boleta de calificaciones
  • Correo electrónico institucional verificado

Requisitos de los documentos

Asegúrate de que tu documentación cumpla con estos requisitos:

  • Debe mostrar claramente tu nombre
  • Debe incluir fecha de emisión o vigencia
  • Debe mostrar el nombre de la institución
  • Debe estar en un idioma que GitHub pueda verificar
  • Las imágenes deben ser nítidas y legibles

    Solicitar GitHub Education

    Una vez que tengas tu documentación lista, es hora de realizar la solicitud oficial.

    Paso 1: Visita GitHub Education

    Ve a education.github.com y haz clic en "Get student benefits".

    Paso 2: Inicia sesión

    Inicia sesión con tu cuenta de GitHub. Si no tienes una, créala primero.

    Paso 3: Completa el formulario

    Proporciona información sobre tu institución y programa de estudios.

    Paso 4: Sube documentación

    Sube los documentos que preparaste en el paso anterior.

    Paso 5: Espera la verificación

    GitHub revisará tu solicitud. Esto puede tomar desde horas hasta algunos días.

      Recursos adicionales

      Además del Student Developer Pack, GitHub Education ofrece otros recursos valiosos para estudiantes.

      GitHub Classroom

      Una herramienta para educadores que simplifica la distribución de tareas y la recolección de código.

      • Creación automática de repositorios para tareas
      • Integración con sistemas de calificación

      • Entrenamiento en liderazgo comunitario

      • Red global de campus experts
